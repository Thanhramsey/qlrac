## 4) Chạy ở máy nhà

### Cài dependency
```bash
npm install
npm install --prefix admin-frontend
```

### Seed menu và dữ liệu nền
```bash
npm run db:seed
```

### Build backend
```bash
npm run build
```

### Build frontend
```bash
npm run build --prefix admin-frontend
```

## 4.1) Đồng bộ dữ liệu DB sang máy nhà

### File backup đã tạo hôm nay
- `backups/db-export-20260703-170416.json`

### Cách export DB ở máy hiện tại (nếu cần tạo bản mới)
```bash
npm run db:export
```

### Cách import DB ở máy nhà
```bash
npm run db:import
```

Ghi chú:
- Lệnh `db:import` mặc định lấy file backup mới nhất trong thư mục `backups/`.
- Hoặc chỉ định file cụ thể:
```bash
node scripts/db-import.mjs backups/db-export-20260703-170416.json
```

### Chạy dev đồng thời
```bash
npm run dev:all
```
